# 돈키우기 (GrowMoney) — Product Requirements

## Problem
투자에 관심은 있지만 실제로 시작하기 어렵고, 금융 공부는 지루하다.

## Target Users
토스 앱 사용 2030대 직장인/학생 — 금융에 관심 있지만 아직 투자 경험이 적은 사람

## Core Features (MVP — Phase 1)

### 1. 오늘의 투자 선택
- 매일 투자 선택지 3개 제공 (삼성전자 / 미국채 / 금 등 mock 데이터)
- 하나 선택 → 다음 날까지 결과 대기
- 이미 선택한 날은 "오늘의 선택 완료" 상태 표시

### 2. 가상 자산 현황
- 시작 자산: 가상 10,000원
- 선택 이력 기반 자산 변동 (mock 데이터로 시뮬레이션)
- 누적 수익률 % 표시

### 3. 결과 카드 (공유용)
- 오늘의 결과 확인 시 공유 카드 자동 생성
- 카드 내용: 자산 현황, 수익률, "상위 N%" 텍스트
- 카카오톡 공유 버튼

### 4. 게임화 요소 (기본)
- 스트릭: 연속 투자 일수 표시
- 누적 수익률 히스토리 (간단한 차트)

## Out of Scope (MVP)
- 실제 시장 API 연동 (Phase 2)
- 랭킹 순위 (Phase 2)
- 친구 대결 (Phase 2)
- 광고/결제 수익화 (Phase 2)
- 토스 SDK 실제 연동 (Phase 2)
- 뱃지/레벨 시스템 (Phase 2)

## Success Criteria
- 핵심 플로우 (선택 → 결과 → 공유) end-to-end 작동
- CEO "이게 내가 원하던 거야" 확인
- Essence 점수 >= 7/10 (critic 리뷰)
- 공유 카드 이미지 생성 성공률 99%+
